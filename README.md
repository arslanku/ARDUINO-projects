# Открытый репозиторий реализованных проектов на платформе Arduino/ESP

Всего за активное время работы было создано 45 технических проектов, некоторые были скрыты в рамках соглашения с заказчиком

Основные направления разработки:
* автоматизированные системы управления, в т.ч. промышленным оборудованием, моторами, реле, сервоприводами, модули контроля времени работы сотрудников
* системы Умного дома (Яндекс Алиса и др), сигнализации, контроль работы обогрева загородных домов, получение показаний с датчиков (температура, давление, влажность...)
* системы дистанционного управления с помощью радиомодулей 433МГц, 2,4ГГц, использование GSM
* управление светодиодными лентами (в т.ч. адресными), источниками света, костюмы для выступлений артистов на сцене
* работа с купюроприемниками, вендинговыми автоматами
* разработка ПО для Windows с функцией управления платой
* проекты для дипломных и научных работ
* модификация уже существующих open-source проектов под требование заказчика
